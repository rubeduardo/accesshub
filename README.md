# AccessHub

Plataforma open source para inclusão, acessibilidade e gestão de organizações que atendem pessoas com deficiência.

## Visão

AccessHub é uma plataforma dedicada a promover inclusão e acessibilidade, facilitando a gestão de organizações que atendem pessoas com deficiência.

## Funcionalidades

- Gestão de organizações
- Controle de acessibilidade
- Sistema de inclusão digital
- Interface acessível

## Começando

### Pré-requisitos

- Node.js
- npm ou yarn
- Docker (opcional)

### Instalação

```bash
git clone https://github.com/rubeduardo/accesshub.git
cd accesshub
npm install
```

### Desenvolvimento

```bash
npm run dev
```
